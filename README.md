Personal webpage developed to showcase a bit about myself, my skills, interests, and background. A modern project built with Next.js, React, TypeScript and Tailwind CSS, focused on performance, responsive design and user experience.(en)

## 🚀 Como Executar o Projeto

### Pré-requisitos
- Node.js (versão 16 ou superior)
- npm ou yarn
- Git

### Instalação e Execução

1. **Clone o repositório:**
   ```bash
   git clone [URL_DO_REPOSITORIO]
   ```

2. **Entre no diretório do projeto:**
   ```bash
   cd dev-portfolio
   ```

3. **Instale as dependências:**
   ```bash
   npm install
   # ou
   yarn install
   ```

4. **Execute o projeto em modo de desenvolvimento:**
   ```bash
   npm run dev
   # ou
   yarn dev
   ```

5. **Acesse no navegador:**
   Abra [http://localhost:3000](http://localhost:3000) para ver o resultado.


## 📚 Guia Git 

### Fluxo de Trabalho Recomendado

1. **Clone o repositório:**
   ```bash
   git clone [URL_DO_REPOSITORIO]
   ```

2. **Entre no caminho do projeto:**
   ```bash
   cd dev-portfolio
   ```

3. **Atualize o repositório local com as últimas mudanças:**
   ```bash
   git pull
   ```

4. **Crie uma nova branch para sua feature/correção:**
   ```bash
   git checkout -b "nome/oqueestafazendo"
   ```
   > **Exemplo:** `git checkout -b "pedro/adicionar-secao-projetos"`

5. **Faça suas alterações no código e adicione os arquivos:**
   ```bash
   git add .
   # ou para arquivos específicos:
   git add nome-do-arquivo.ext
   ```

6. **Commit suas mudanças:**
   ```bash
   git commit -m "AB#(ID DA TASK DO AZURE) algum comentário se quiser..."
   ```
   > **Exemplo:** `git commit -m "AB#123 Adiciona seção de projetos no portfólio"`

7. **Envie sua branch para o repositório remoto:**
   ```bash
   git push -u origin nome/oqueestafazendo
   ```

### 💡 Dicas Importantes para Iniciantes

- **Sempre crie uma nova branch** para cada feature ou correção
- **Faça commits pequenos e frequentes** com mensagens descritivas
- **Teste seu código** antes de fazer push
- **Mantenha sua branch atualizada** com `git pull` regularmente

### 🆘 Comandos Úteis

```bash
# Ver status dos arquivos
git status

# Ver histórico de commits
git log --oneline

# Voltar para a branch principal
git checkout main

# Deletar uma branch local
git branch -d nome-da-branch

# Ver todas as branches
git branch -a
```
